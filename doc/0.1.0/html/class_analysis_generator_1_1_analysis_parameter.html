Detailed Description

Class representing a parameter used for analysis.

This class represent a single parametr used for analysis. Each parameter has a name (member name_). Than it is possible to give bounds for parameter's values (members lb_ and ub_) or a set of allowed values (member values_). Last feature of a parameter is being scalar or vectorial (one dimensional array). Such characteristic is defined via member dim_. It exposes only member methods devoted to information retriving. All constructors and setting method are private and accessible only by class AnalysisParameter which handles a set of AnalysisParameter instances
